# Profile-card-project
A modern and responsive (mobile-friendly) profile card design. Developed using HTML5, CSS3 Flexbox, and Font Awesome.
Modern Profile Card Project

Bu proje, Frontend Developer yolculuğumdaki ilk profesyonel adımımdır. Şık, modern ve kullanıcı etkileşimi yüksek bir profil kartı tasarımıdır.

Bu Projede Neler Öğrendim?

Bu süreci sadece kod yazarak değil, karşılaştığım problemleri teknik yöntemlerle çözerek tamamladım:

Box Model & Border Radius:** HTML elementlerinin kutu yapısını ve köşeleri `border-radius` ile nasıl estetik hale getireceğimi öğrendim.
Harici Kaynak Yönetimi (CDN):** Yerel fontların çalışmadığı durumlarda, Cloudflare üzerinden harici kütüphaneleri (CSS/Font) projeye nasıl güvenli bir şekilde dahil edeceğimi (`href` kullanımı) keşfettim.
CSS Flexbox:Elementleri sayfa üzerinde kusursuz bir şekilde hizalamayı ve aralarındaki boşlukları (`gap` ve `margin`) profesyonelce yönetmeyi öğrendim.
Görsel Optimizasyonu: Profil fotoğraflarının dairesel formda ve doğru orantıda gösterilmesi üzerine çalıştım.
Versiyon Kontrolü (Git/GitHub):Projeyi bilgisayarımdan GitHub'a taşımayı, dosya yönetimini ve README dökümantasyonu oluşturmayı deneyimledim.

Kullanılan Teknolojiler

HTML5:** Semantik etiket yapısı.
CSS3:** Flexbox, Transitions, Hover Effects.
Icons:** Font Awesome (Cloudflare CDN üzerinden).
Fonts:** Google Fonts.

Responsive Yapı
Tasarım, farklı ekran boyutlarında kullanıcı deneyimini bozmayacak şekilde planlanmıştır.


Bu proje Selim Enes Çinçe tarafından 3 aylık yoğun yazılım kampı sürecinin bir parçası olarak geliştirilmiştir.
